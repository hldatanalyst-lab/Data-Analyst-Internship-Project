# Data Analyst Capstone Internship Project

This repository contains my complete end-to-end Data Analyst Internship project.  
It demonstrates my skills across *Excel, SQL, Power BI, Tableau,* and overall data analysis methodology.

---

## Project Structure

### 1. Module 1 – Data Understanding & Excel Work
Includes:
- Raw dataset
- Data cleaning steps
- Exploratory analysis in Excel
- Charts, pivot tables, and summary insights

### 2. Module 2 – SQL Analysis
Contains:
- 24 SQL queries
- All module-wise assignments
- Joins, subqueries, window functions, aggregations

### 3. Module 3 – Power BI Dashboard
Includes:
- PBIX dashboard file
- Data model
- DAX measures
- KPI metrics & visual insights

### 4. Module 4 – Tableau Visualization
Contains:
- TWBX packaged workbook
- Final dashboard screenshots
- LOD expressions & calculations

---

## Tools Used

- **Excel** (Cleaning, EDA, Dashboards)
- **SQL** (MySQL / PostgreSQL)
- **Power BI** (DAX, Data Modeling, Visualization)
- **Tableau** (LOD, Interactive dashboards)

## Project Goal

To perform full-cycle data analysis:

4. Build BI dashboards (Power BI & Tableau)  
5. Deliver final insights and recommendations
